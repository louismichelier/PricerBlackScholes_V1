# PricerBlackScholes_V1_C++
Ce projet est un moteur de calcul haute performance pour les options européennes de 50 lignes codé à la main en C++


Exemple de retour : 
========= PRICER BLACK-SCHOLES =========
Prix du Call : 10.4506 EUR
Prix du Put  : 5.5735 EUR
----------------------------------------
Delta (Hedging) : 0.6368
Strategie : Achetez 0.6368 actions pour couvrir 1 Call.
========================================
